# 軟體工程教學github用法
設定 Git 身分
```bash
git config --global user.name "你的名稱"
git config --global user.email "你的信箱"
```

更新
```bash
git pull
```

查詢分支目錄
```bash
git branch -a
```

顯示遠端分支的列表
```bash
git branch -r
```

建立新分支
```bash
git branch -b hong/button
```

進入分支
```bash
git switch hong/button
```

合併分支
```bash
git merge zzh/button
```
